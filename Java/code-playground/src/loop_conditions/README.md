# loop_conditions – Java Grundlagen

Dieses Projekt enthält einfache Java-Beispiele zu Operatoren, Schleifen, Arrays und Switch-Anweisungen.  
Der Fokus liegt auf grundlegender Syntax, Verständnis von Kontrollstrukturen und sauberer Trennung von Aufgaben.

## Klassenübersicht

**BasicsOperators**  
Demonstriert den Einsatz von Inkrement-Operatoren (`++`) und deren Auswirkungen auf Variablen.

**ForLoops**  
Enthält klassische `for`-Schleifen wie Zählerschleifen, Summenberechnung, Countdown und Multiplikationstabellen.

**ArraysDemo**  
Zeigt die Arbeit mit Arrays, Index-Zugriff, `foreach`-Schleifen und typische Anfängerfallen.

**WhileDoWhile**  
Beispiele für `while`- und `do-while`-Schleifen inklusive Benutzereingaben mit `Scanner`.

**SwitchDemo**  
Demonstriert `switch`-Anweisungen mit Ganzzahlen und Strings anhand von Wochentagen, Quartalen und Rollen.

**Calculator**  
Ein einfacher Taschenrechner mit Methodenüberladung und `switch`-basierter Operatorauswahl.

**WhileConditionTrap**  
Zeigt eine komplexe `while`-Bedingung und erklärt, warum sie nicht durch ein `if` ersetzt werden kann.

## Ziel des Projekts

Dieses Projekt dient als Übungsmaterial für Java-Grundlagen und zur Vorbereitung auf Prüfungen und weiterführende Themen.
