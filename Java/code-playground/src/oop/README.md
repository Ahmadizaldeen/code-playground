# OOP Demo

Dieses package enthält Java-Beispiele zu den Themen:

- Klassen, Konstruktoren und Methoden (`Produkt.java`)
- Interfaces und Vererbung (`VererbungUndInterfacesDemo.java`)
- String-Manipulation mit Interfaces (`StringInterfaceDemo.java`)

Die Interfaces `MyInterface` und `Beweglich` sind in einem eigenen Paket `oop.interfaces`.

---

## Struktur
- `Produkt.java` – Demonstration von Klassen, Konstruktoren und Methoden
- `StringInterfaceDemo.java` – Interfaces und String-Verarbeitung
- `VererbungUndInterfacesDemo.java` – Vererbung, Interfaces, Tier- und Geräte-Klassen
- `interfaces/` – Enthält alle Interfaces

oop/
├── Produkt.java
├── StringInterfaceDemo.java
├── VererbungUndInterfacesDemo.java
└── interfaces/
    ├── MyInterface.java
    └── Beweglich.java
