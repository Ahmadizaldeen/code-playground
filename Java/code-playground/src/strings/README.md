# Java Strings Project

Dieses Projekt zeigt Beispiele für:

- String Pool & `intern()`
- Bitoperationen: `<<`, `>>`, `>>>`, Flags
- Benutzerinteraktion mit Strings (`length`, `charAt`, `equals`, `substring`, `replace`, etc.)
- StringBuilder Operationen (`append`, `insert`, `replace`, `delete`, `reverse`)
- Vergleich von Strings (`compareTo`, `compareToIgnoreCase`)
- Grundlegende String-Methoden (`trim`, `split`, `toUpperCase`, `toLowerCase`)

## Klassen

- **Main**: Einstiegspunkt, zeigt grundlegende Beispiele
- **Aufgaben**: Bitoperationen und Übungen
- **StringOperations**: Methoden für interaktive String Operationen
- **StringBuilderUtils**: Beispiele mit `StringBuilder` und Vergleich
- **StringKlasseUtils**: Basis String Methoden, trim, substring, replace
